Como ler esta página (em uma respirada)
Essa página mostra os números da Égide Saúde & Benefícios — que é tipo um "iFood da saúde": clínicas, médicos e farmácias se cadastram, e os clientes marcam consulta, exame ou compram remédio pelo site/app. A Égide é da CAMIM, mas tem banco de dados separado dos postos.
  • Consulta = cliente vai num médico de uma especialidade (cardiologia, oftalmo…).
  • Exame = ele vai fazer um exame (sangue, raio-X, ultra…) — ainda é agendamento, mas sem médico fixo.
  • Particular = cliente pagou do próprio bolso (cartão, PIX, boleto).
  • Convênio = o plano de saúde dele (Bradesco, Unimed, SulAmérica…) está cobrindo. Ele não paga direto; quem paga é o convênio.
⚠️ Dica pra não se confundir: Quando a gente fala "receita", é dinheiro que entrou. Quando a gente fala "consulta paga", é quando o cliente já quitou o valor (não só agendou). Uma consulta agendada pode ser cancelada depois — por isso os números de agendada e paga são diferentes.
Os dados começam em janeiro/2023. O mês em andamento aparece com PARCIAL porque ainda não fechou — é só uma prévia.
Ctrl/Cmd + clique para selecionar vários. Vazio = todos.

1. Clientes por período

O que cada número significa
  • Base total = todo mundo que já se cadastrou na Égide desde o começo (mesmo quem nunca mais voltou).
  • Novos no mês = quantas pessoas se cadastraram pela primeira vez naquele mês (conta abriu).
  • Ativos no mês = quantas pessoas usaram de verdade a Égide naquele mês (pagaram uma consulta ou um pedido de farmácia).
💡 Comparação útil: Se os "novos" crescem mas os "ativos" ficam iguais, quer dizer que chega gente nova mas os antigos somem. O ideal é os dois subindo juntos.

2. Consultas e exames por período

Como ler este gráfico
Cada barra é um mês. A barra é dividida em dois:
  • Consulta (azul) — quando o cliente vai num médico de uma especialidade (clínico, pediatra, cardiologista…).
  • Exame (laranja) — quando o cliente vai fazer um exame (raio-X, sangue, ultrassom…). Não tem médico fixo, ele só faz o exame.
Logo abaixo mostra também quantas foram pagas e quantas foram canceladas — porque nem tudo que é agendado acontece de verdade.
⚠️ Dado importante: a taxa de consultas pagas subiu muito ao longo de 2025/2026 (de ~20% pra ~75%). Isso provavelmente é porque a Égide passou a cobrar antes pra confirmar o agendamento.

3. Particular vs Convênio

Qual a diferença?
  • Particular = cliente pagou do próprio bolso (cartão, PIX, boleto). A Égide recebe o dinheiro na hora.
  • Convênio = o plano de saúde do cliente (Unimed, Bradesco, SulAmérica…) está cobrindo. A Égide cobra depois, do convênio, não do cliente.
Pra saber se uma consulta é de convênio, a gente checa se o cliente tinha um convênio ativo naquele dia (tabela customer_insurances).
📊 O que você vai ver: a pizza mostra o total desde o começo — e deixa na cara que a maior parte da receita é particular. Convênio, por enquanto, é uma fatia pequena.

4. Arrecadação por contrato (Particular)

O que é "arrecadação por contrato"?
É o dinheiro que a Égide recebe direto do cliente (sem convênio no meio). Vem de três caminhos:
  • Clínica — cliente pagou uma consulta ou exame.
  • Farmácia — cliente comprou remédio ou produto pelo app.
  • Outros — outras vendas do marketplace (kits, cortesias, telemedicina avulsa).
💡 Leitura do gráfico: cada barra é um mês, empilhado pelos três caminhos. Dá pra ver se a farmácia cresceu, se a clínica está puxando, etc.

5. Arrecadação por Convênio

Como funciona esse gráfico
Cada barra é um mês, e dentro dela aparecem faixinhas coloridas: uma cor por convênio (Unimed, Bradesco, SulAmérica, Colaboradores OC, Égide Promocional, etc.).
  • Se a faixinha aumenta de um mês pro outro, aquele convênio está trazendo mais receita.
  • Se a faixinha some, aquele convênio parou de mandar cliente naquele mês.
  • Quanto mais colorida a barra estiver, mais convênios diferentes estão contribuindo.
📌 Importante: use o filtro "Convênios" lá em cima pra olhar só o(s) convênio(s) que te interessam. Deixando vazio, aparecem todos.
